
# 🌍 AfriConnect Summit 2026

Site web dynamique pour l'événement AfriConnect Summit 2026.

## 📌 Présentation du Projet
AfriConnect Summit 2026 est un site web multi-pages dynamique dédié à la promotion de l'innovation technologique et de la jeunesse en Afrique.

## 🚀 Fonctionnalités Intégrées
- **Navigation Responsive :** Menu hamburger fonctionnel pour mobiles et tablettes.
- **Thème Sombre / Clair :** Bascule dynamique intégrée via JavaScript.
- **Formulaire d'Inscription Sécurisé :** Validation complète côté client (JS) avec gestion des messages d'erreur.
- **FAQ Interactive :** Accordéon dynamique pour les questions fréquentes.
- **Animations & Interactivité :** Fil d'ariane, intégration Google Maps et retour en haut de page.

## 🛠️ Technologies Utilisées
- **HTML5** (Sémantique et accessible)
- **CSS3** (Flexbox, Grid, CSS Variables)
- **JavaScript** ( Vanilla )
- **Bootstrap Icons**
